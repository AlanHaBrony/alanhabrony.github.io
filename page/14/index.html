9008 and 9006

2024-03-13 20:07
After searching, I noticed that my wt88047 was only in 9006 Download mode, so I used FTK to find which partion is “sdl1” and deleted it to successfully into 9008 edl mode.
However, Miflash threw error “couldn’t read hello packet” and QFIL threw error”function: main320 Uploading Image using Sahara protocol failed”.
I searched again, but taking off the battery and reconnected the device are all no use.
And I did not find any guide or explanation.
Since I don’t have any data in wt88047, so I decide to change another device and back to it when I learnded more.

One thing to note is:
9006 mode could be read as a Disk and copy partions directly, while 9008 mode could not be recognized as Disk and I don’t know how to switch from 9008 to 9006. So be careful when switching to 9008.
